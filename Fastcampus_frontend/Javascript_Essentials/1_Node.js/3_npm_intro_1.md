### NMP이란?

NPM(Node Package Manager)은 전세계의 개발자들이 만든 <br/>다양한 기능 패키지, 모듈)들을 관리.

![](https://images.velog.io/images/sh981013s/post/2b893809-75a0-400a-aab0-9bb04b7e0f79/image.png)

가운대의 영역이 npm 생태계이다.
```
$npm install XXX
```
를 통하여 모듈을 로컬 패키지로 설치할 수 있다.

전세계의 개발자들이 모듈, 기능들을 개발해 생태계로 업로드하는 시스템이다.

![](https://images.velog.io/images/sh981013s/post/966e462c-e99b-45d6-bd37-e7d13218393f/image.png)

하지만 결국에는 node.js 라는 환경에서 npm 이라는 개념으로 패키지들을 설치하고 관리하고 동작시켜야하는데 해당 방식에서는 각각의 패키지들을 이해하고 학습을 하고 사용해야한다. 

그럼에도 불구하고 해당 방식을 사용하는 이유는 해당 프로젝트를 관리할때 효율도 증가하고, 남이 만들어 놓은 기능을 사용하기 때문에 더욱 손쉬워진다.
