# 웹앱의 동작원리

### 1. 최초 요청 (Request)
최초에 웹 브라우저로 주소를 입력하면 서버에 최초 요청이 들어간다.

### 2. 최초 응답 (Response)
화면에 기본적인 HTML 구조로 응답한다.
### 3. 추가 요청 
이미지 파일 등 추가적인 정보를 요청한다.
### 4. 추가 응답

서버에서는 추가적인 CSS,JS,JPG 등 추가적인 정보로 응답한다.
