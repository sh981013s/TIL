# CSS Box Model



## h1
	h1 태그는 가로폭 "전체" 쓴다.

## a
	a태그는 자신의 "부피" 만큼을 쓴다. 


#### 화면 전체를 사용하는 태그는 "block level element" , 

#### 자신의 필요한 부피만큼을 사용하는 태그는 "inline element" 라고 한다.
